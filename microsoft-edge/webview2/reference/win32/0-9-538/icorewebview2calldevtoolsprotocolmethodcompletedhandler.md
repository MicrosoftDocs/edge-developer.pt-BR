# interface ICoreWebView2CallDevToolsProtocolMethodCompletedHandler 

```
interface ICoreWebView2CallDevToolsProtocolMethodCompletedHandler
  : public IUnknown
```

O chamador implementa essa interface para receber os resultados da conclusão do CallDevToolsProtocolMethod.
