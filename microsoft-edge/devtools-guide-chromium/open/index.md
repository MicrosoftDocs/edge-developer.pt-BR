## <span data-ttu-id="4e817-134">Abrir automaticamente o DevTools em cada nova guia</span><span class="sxs-lookup"><span data-stu-id="4e817-134">Auto-open DevTools on every new tab</span></span>

### [<span data-ttu-id="4e817-136">CMD (Windows)</span><span class="sxs-lookup"><span data-stu-id="4e817-136">CMD (Windows)</span></span>](#tab/cmd-Windows/)  

<a id="auto-open-devtools-command-line"></a>  

```cmd

```  

### [<span data-ttu-id="4e817-137">PowerShell (Windows)</span><span class="sxs-lookup"><span data-stu-id="4e817-137">PowerShell (Windows)</span></span>](#tab/powershell-Windows/)

Start-Process -FilePath "msedge" -ArgumentList "--auto-open-devtools-for-tabs"
```

```
