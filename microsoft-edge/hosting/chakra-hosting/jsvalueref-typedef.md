# Typedef JsValueRef
Uma referência a um valor de JavaScript.  
  
## Sintaxe  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## Comentários  
 Um valor JavaScript é um dos seguintes tipos de valores: undefined, NULL, Boolean, String, Number ou Object.  
  
## Requisitos  
 **Header:** jsrt. h
