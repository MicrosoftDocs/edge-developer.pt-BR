### <span data-ttu-id="43b02-122">Cada guia do navegador é isolada de todas as outras guias</span><span class="sxs-lookup"><span data-stu-id="43b02-122">Each browser tab is isolated from every other tab</span></span>

### <span data-ttu-id="43b02-126">Cada guia manipula uma solicitação GET</span><span class="sxs-lookup"><span data-stu-id="43b02-126">Each tab handles one GET request</span></span>  

<span data-ttu-id="43b02-127">Cada guia basicamente usa a URL \ (também conhecida como o localizador de recursos uniforme \) para obter um único fluxo de dados, que normalmente é um documento HTML.</span><span class="sxs-lookup"><span data-stu-id="43b02-127">Each tab essentially uses the URL \(also known as the uniform resource locator\) to get a single stream of data which is typically an HTML document.</span></span>  <span data-ttu-id="43b02-128">Esse único fluxo \ (ou página \), muitas vezes inclui instruções \ (como JavaScript incluem marcas, referências de imagem, referências CSS e mais \).</span><span class="sxs-lookup"><span data-stu-id="43b02-128">That single stream \(or page\), often includes instructions \(like JavaScript include tags, image references, CSS references, and more\).</span></span>  <span data-ttu-id="43b02-129">Em última análise, todos os recursos necessários são baixados para uma única página da guia e normalmente é exibida uma visualização que vemos na guia do navegador completamente renderizada.</span><span class="sxs-lookup"><span data-stu-id="43b02-129">Ultimately, all the resources needed are downloaded to that one tab page and typically a visualization appears which we see in the browser tab completely rendered.</span></span>  

### <span data-ttu-id="43b02-130">Todas as comunicações de cada guia são para servidores remotos</span><span class="sxs-lookup"><span data-stu-id="43b02-130">All communication from each tab is to remote servers</span></span>

## <span data-ttu-id="43b02-133">O modelo de extensão transforma tudo de cabeça para baixo</span><span class="sxs-lookup"><span data-stu-id="43b02-133">The Extension model turns everything upside down</span></span>  

<span data-ttu-id="43b02-134">Uma extensão, exatamente como as páginas da guia, é executada em um thread individual que é completamente isolado de todos os threads de página da guia que são abordados.</span><span class="sxs-lookup"><span data-stu-id="43b02-134">An Extension, just like tab pages, runs in a individual thread which is completely isolated from all tab page threads that are discussed.</span></span>  <span data-ttu-id="43b02-135">Ao contrário das guias cujo trabalho geralmente emite uma única solicitação GET para um servidor remoto e, em seguida, exibe uma visualização desses dados no navegador, a extensão, por outro lado, é o servidor, que anteriormente residau na outra extremidade da conexão à Internet feita a partir de uma guia do navegador.</span><span class="sxs-lookup"><span data-stu-id="43b02-135">Unlike the tabs whose job is to typically issue a single GET request to a remote server, then display a visualization of that data in the browser, the Extension, on the other hand is the server, that previously resided on the other end of the internet connection made from a browser tab.</span></span>  

![O modelo de extensão transforma o modelo de servidor de cabeça para baixo](media/index-image3-upsidedown.png)  

<span data-ttu-id="43b02-137">Isso é muito importante compreender.</span><span class="sxs-lookup"><span data-stu-id="43b02-137">This is really important to understand.</span></span>  <span data-ttu-id="43b02-138">Depois de criar uma extensão e instalá-la no seu navegador, você criou um servidor Web autônomo que está morando e livrendo dentro do seu navegador, mas ainda isolado de cada página da guia em execução nesse navegador.</span><span class="sxs-lookup"><span data-stu-id="43b02-138">Once you create an Extension, and install it in your browser, you've created a standalone web server that is living and breathing inside of your browser but still isolated from every tab page running on that browser.</span></span>  

### <span data-ttu-id="43b02-139">O pacote do servidor Web de extensão</span><span class="sxs-lookup"><span data-stu-id="43b02-139">The Extension web server bundle</span></span>  

<span data-ttu-id="43b02-140">Então, o que é uma extensão?</span><span class="sxs-lookup"><span data-stu-id="43b02-140">So what is an Extension?</span></span> <span data-ttu-id="43b02-141">É um pacote \ (ou conhecido como um arquivo zip \) de recursos da Web que não são diferentes dos que um desenvolvedor da Web publica em um servidor Web.</span><span class="sxs-lookup"><span data-stu-id="43b02-141">It is a bundle \(or referred to as a zip file\) of web resources that are no different than what a web developer publishes to a web server.</span></span>  

<span data-ttu-id="43b02-142">Esse arquivo zip inclui HTML, CSS, JavaScript, imagens e todos os ativos necessários para criar uma página da Web.</span><span class="sxs-lookup"><span data-stu-id="43b02-142">That zip file includes HTML, CSS, JavaScript, images and all the necessary assets to make a web page.</span></span>  <span data-ttu-id="43b02-143">No entanto, há um arquivo extra que é necessário na raiz desse arquivo zip e esse arquivo é nomeado `manifest.json` .</span><span class="sxs-lookup"><span data-stu-id="43b02-143">There is however, one extra file that is required in the root of this zip file, and that file is named `manifest.json`.</span></span>  <span data-ttu-id="43b02-144">É o esquema da extensão que inclui itens como qual é a versão de sua extensão, qual é o título, em que privilégios ele precisa ser executado e muito mais.</span><span class="sxs-lookup"><span data-stu-id="43b02-144">It is the blueprint for your Extension that includes things like what is the version of your Extension, what is the title, what privileges does it need to run and lots more.</span></span>
