# Função JsCreateSymbol

Cria um símbolo JavaScript.
  
## Sintaxe  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateSymbol(  
   _In_ JsValueRef description,  
   _Out_ JsValueRef *result  
);  
```  
  
#### Parâmetros  
 `description`  
 A descrição da cadeia de caracteres do símbolo. Pode ser nulo.  
  
 `result`  
 O novo símbolo.  
  
## Valor de retorno  
 O código `JsNoError` se a operação tiver sido bem-sucedida, um código de falha também.  
  
## Comentários  
 Requer um contexto de script ativo.  
  
 Essa API só tem suporte no modo Microsoft Edge.  
  
## Requisitos  
 **Header:** jsrt. h
